FEATURE GAP ANALYSIS

VTuber 리깅 검수 도구
기능 격차 분석 리포트

유사 서비스·프로그램(VTube Studio, nizima LIVE, Live2D Cubism, Warudo, Frame.io 등)에는 있지만 현재 앱에는 아직 없는 기능들을 모아, 우선순위와 함께 정리했습니다.
대상 앱 버전 v1.7.2 2026.06.30 작성 Live2D / Cubism4 · Next.js

비교 대상 · VTube Studio / nizima LIVE / Live2D Cubism Editor·Viewer / Warudo / VSeeFace / Inochi2D / Frame.io · Figma

현재 앱이 이미 갖춘 기능 baseline

업로드·갤러리(폴더 구조 보존, 버전별 파일 목록) · 드라이브 백업(임의 파일/폴더, 갤러리 등록) · 파라미터 조작(슬라이더, 극한값 자동 스윕 테스트) · 아트메쉬(클릭 선택, 그룹 on/off, 숨김 공유 저장) · 연출(모션/표정/배경/아이들/프리즈/스크린샷) · 시선 포인터(보는 방향 표시) · 코멘트(상태 첨부·복원, 삭제) · 버전 이동(모바일 버튼 / PC 드래그) · 딥링크 상태 공유. 이 리포트는 위 기반 위에서 "비교 대상에는 있는데 우리에겐 없는" 기능만 추립니다.

1. 실시간 트래킹 · 라이브 tracking & live

VTube Studio·nizima LIVE·VSeeFace의 핵심. 검수 시 "실제로 움직였을 때" 리깅을 확인하게 해줍니다.
웹캠 얼굴 트래킹우선순위 중VTube Studio · nizima LIVE · VSeeFace
웹캠으로 눈·입·고개 움직임을 모델에 실시간 반영. 검수자가 직접 표정을 지어 파라미터 연동을 즉석에서 확인. MediaPipe FaceMesh(브라우저)로 구현 가능.
마이크 립싱크우선순위 중VTube Studio · Animaze
마이크 음량/모음을 입 모양 파라미터로. MouthOpen·MouthForm 리깅이 말할 때 자연스러운지 검수. Web Audio API만으로 음량 기반 버전은 즉시 가능.
iOS ARKit / 폰 트래킹 연동우선순위 하VTube Studio · iFacialMocap
아이폰 트루뎁스 52개 블렌드셰이프 수신. 고품질 트래킹이지만 네이티브 의존도가 높아 후순위.

2. 리깅 검수 특화 rigging review

"리깅 검수 도구"라는 정체성에서 가장 차별화가 큰 영역. 우선순위가 높습니다.
두 버전 나란히 비교 (Before/After)우선순위 상Frame.io · Figma
같은 모델의 v1·v2를 좌우 분할로 동시 표시하고 파라미터를 동기 조작. 수정 전후 차이를 한눈에. 이미 버전 개념이 있어 가장 자연스러운 확장.
원화 오버레이 · 어니언 스킨우선순위 상Live2D Cubism · 작화 검수
원본 일러스트를 반투명으로 겹쳐, 리깅으로 형태가 무너졌는지(라인·실루엣 이탈) 확인. 이미지 업로드 + opacity 슬라이더로 구현 단순.
위치 고정 코멘트(핀)우선순위 상Frame.io · Figma · Vimeo Review
"여기 입꼬리"처럼 모델의 특정 좌표(또는 메쉬)에 핀으로 코멘트. 현재 코멘트는 위치 정보가 없음. 메쉬 picking이 이미 있어 좌표/메쉬 앵커 저장만 추가하면 됨.
승인 워크플로 (상태 라벨)우선순위 중Frame.io · 디자인 리뷰 툴
버전마다 검토중 / 수정요청 / 승인됨 상태. 누가 OK했는지 기록되어 작업 핑퐁이 줄어듦.
메쉬 관통·클리핑 자동 검출우선순위 하Cubism 검증 · 자체
극한값 스윕 중 메쉬 경계가 비정상적으로 벗어나는 프레임을 자동 감지해 표시. 이미 만든 극한값 테스트의 확장형.

3. 출력 · 내보내기 output

프리뷰 영상 / GIF 녹화우선순위 중대부분 VTuber 앱 · OBS
현재는 스크린샷(정지)만 가능. 움직이는 검수 결과를 영상/GIF로 공유. 브라우저 MediaRecorder + canvas.captureStream으로 구현 가능.
투명 배경 / 가상카메라(OBS) 출력우선순위 하VTube Studio · Spout2/Syphon
방송용 합성 출력. 웹앱 특성상 가상카메라는 한계가 있으나, 투명 PNG/WebM 내보내기는 가능.
모델 성능 통계우선순위 중Live2D Cubism · 성능 패널
아트메쉬 수·정점 수·텍스처 용량·드로콜·마스크 개수를 표로. 모바일에서 무거운 모델인지 검수자가 판단. 이미 로딩한 데이터로 산출 가능.

4. 협업 · 계정 collaboration

코멘트 알림 (이메일/푸시)우선순위 상Frame.io · Figma · GitHub
새 코멘트가 달리면 작업자에게 알림. "확인했나?"를 따로 챙기지 않아도 됨. Supabase 트리거 + 웹훅/이메일로 구현.
코멘트 스레드 · @멘션 · 반응우선순위 중Figma · Frame.io
답글 스레드, @멘션, 이모지(👍/✅) 반응, 해결됨 처리. 현재는 평면 코멘트만. 토론 맥락이 살아남.
계정 · 권한 · 비공개 갤러리우선순위 중대부분의 협업 SaaS
현재는 공개·비번 1개 구조. 로그인/팀/비공개 모델이 있으면 클라이언트 작업물 분리 가능. Supabase Auth로 단계적 도입.
실시간 멀티 커서 · 동시 검수우선순위 하Figma · Figjam
여러 명이 같은 모델을 동시에 보며 커서·선택을 공유. 임팩트는 크지만 구현 비용이 높아 후순위.

5. 라이브러리 관리 · 편의 library & UX

태그 · 검색 · 필터우선순위 상모든 자산 관리 툴
모델이 늘면 필수. 이름·태그·날짜로 검색/정렬. 갤러리에 검색창과 태그 컬럼만 추가하면 큰 효과.
버전 히스토리 · 파일 diff우선순위 중Git · 디자인 버전관리
버전 간 어떤 파일(텍스처·모션·물리)이 바뀌었는지 목록 비교. 이미 버전별 파일 목록이 있어 확장 용이.
표정·모션 단축키(핫키)우선순위 중VTube Studio
키보드 1·2·3으로 표정/모션 즉시 전환. 검수 중 반복 클릭을 줄여줌.
실행 취소 / 다시 실행 · 다국어우선순위 하일반 앱 표준
파라미터·그룹 편집 undo/redo, 영어/일본어 UI(해외 리거 협업 시). 점진 도입 항목.

추천 도입 순서 roadmap

단계기능이유 / 메모
1차 (빠른 효과)두 버전 나란히 비교 · 원화 오버레이 · 위치 고정 코멘트 핀 · 갤러리 검색/태그기존 기능(버전·메쉬 picking·코멘트·갤러리)을 그대로 확장 → 비용 대비 검수 가치가 가장 큼
2차 (협업 강화)코멘트 알림 · 승인 워크플로 · 코멘트 스레드/멘션 · 영상·GIF 녹화 · 성능 통계여러 사람이 쓰는 도구로 성숙. 대부분 Supabase + 브라우저 API로 가능
3차 (라이브)웹캠 얼굴 트래킹 · 마이크 립싱크 · 핫키"움직임 검수" 단계. MediaPipe·Web Audio로 단계적
4차 (고급)계정·권한 · 실시간 멀티커서 · ARKit 연동 · OBS 출력구현 비용이 큰 항목. 사용자 규모가 커질 때
참고 · 우선순위는 "현재 코드 자산으로 얼마나 적은 비용에 큰 검수 가치를 내는가"를 기준으로 매겼습니다. 특히 두 버전 비교 · 원화 오버레이 · 위치 고정 코멘트는 이미 보유한 버전·메쉬 picking·코멘트 인프라를 재활용하므로, 다른 트래킹/계정 기능보다 먼저 넣을 때 투자 대비 효과가 가장 큽니다.
